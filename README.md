# LinkSmth jQuery Plugin

**LinkSmth** is a simple jQuery plugin to attach links to any elements

## Version
0.2.0

## Usage
    $('span').linkSmth({
        href: "http://test.com",
        text: "Click Me",
        target: "_blank"
    });

## Vendors
jQuery - [http://jquery.com](http://jquery.com)
